Zip() in for loops

You have three variables: interest_rates, years and loan_principles. They contain lists of integers. Write a for loop statement that prints the total interest paid for each loan as an integer, each one in a new line.

Use the following formula: interest_paid = interest_rate * year * loan_principle

The following lists interest_rates, years and loan_principles have already been defined; you do not need to accept any input.


Sample Input:
0.02, 0.05, 0.04
2, 3, 5
1000, 2000, 1400

Sample Output:
40
300
280



Memory limit: 256 MB
Time limit: 15 seconds
